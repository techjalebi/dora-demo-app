# OrderFlow

A fictional order management service. This repository is the **simulation target** for the [DORA Reporting Platform](https://github.com/techjalebi/dora-platform) demo project.

Commits, pull requests, and releases in this repo are generated automatically by the DORA simulation engine to produce realistic DevOps activity data across a 6-month window.

## Structure

```
src/        Application source code
tests/      Unit tests
CHANGELOG.md  Release history (updated per simulated release)
```

## Running tests

```bash
pytest tests/
```
